# JavaScript Variables

في هذا الدرس هنتعلم:
- الفرق بين let و const
- كيفية إنشاء المتغيرات
- طباعة القيم باستخدام console.log()

## Example

```javascript
let name = "Aya";
const age = 30;

console.log(name);
console.log(age);
```

## مفاهيم مهمة

- `let` يمكن تغيير قيمتها
- `const` لا يمكن إعادة تعيينها
